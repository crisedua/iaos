## Cómo usar esta lista

1. Identificar contactos que encajan con el ICP (ver criterios abajo)
2. Elegir la variante de script correcta (`reference/challenge-promotion-scripts.md`)
3. Enviar el mensaje
4. Registrar respuesta y estado aquí
5. Hacer seguimiento el Día 3 si no hubo respuesta

**ICP del challenge (perfil objetivo):**
- Dueño/a de negocio, gerente, consultor/a, freelancer, o solopreneur
- Ha dicho alguna vez "no tengo tiempo" o "estoy desbordado"
- Usa ChatGPT o IA de vez en cuando pero no de forma sistemática
- Trabaja principalmente con conocimiento, comunicación, o servicios
