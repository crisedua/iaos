# daily-outreach/

This directory holds per-day outreach queues generated by `scripts/prepare-outreach.js`.

## File naming convention

| File | Contents |
|---|---|
| `YYYY-MM-DD-outreach.json` | Machine-readable queue: array of leads with personalized messages, ICP scores, and contact info |
| `YYYY-MM-DD-outreach.md` | Human-readable review file: one section per lead, formatted for easy review before sending |

## Workflow

```
1. node scripts/prepare-outreach.js        → creates YYYY-MM-DD-outreach.json + .md
2. Review YYYY-MM-DD-outreach.md           → check messages, adjust if needed
3. node scripts/send-whatsapp.js           → sends (or previews) from the JSON queue
```

## Related files

- `outputs/leads/outreach-tracker.json` — master contact log (all contacts across all days)
- `scripts/prepare-outreach.js` — generates these files
- `scripts/send-whatsapp.js` — consumes the JSON queue
- `reference/whatsapp-automation-setup.md` — Twilio setup guide
